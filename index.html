About the Work

This is a face paced course that teaches practical and technical skills that will enable a student to build web applications.

There are multiple programs that are covered that one gain more than just a basic knowledge of over the next 6 months. When the course ends each student will have an impressive portfolio for display for future job aspects.

  • JavaScript
  • Node
  • SQL
  • CSS
  • HTML
  • MongoDB
  • React

By the end of the course you'll be able to:

  • Build front-end websites from scratch
  • Create full-stack single-page web applications
  • Implement structured and unstructured databases to convert static websites into dynamic ones
  • Build communication skills and demonstrate foundational computer-science knowledge
  • Apply the accepted and standard basics of social coding
  • Demonstrate strong teamwork and project management skills as a collaborator and independent contributor during the development cycle of complex projects

Challenge 1 - Horiseon

Challenge 1 was the first homework assignment we turned in for this course. For this assignemnt we were given an already built out code and were asked to modify it with the focus of accessibility. The company Horiseon "hired" us to rework their website to make it more accessible as this is an increasingly important thing for businesses to keep in mind for their webpages. We are given a user story that explains the overall goal of our assignment and then acceptance criteria that are our guidelines of what we need to adjust and update in the code to successfully complete the assignment. For this particular assignment the acceptance criteria was to make sure the code follows a logical strucure, correct semantics were used, and alt text was used appropriately for the photos.

Challenge 2 - Portfolio Build

Challenge 2 is the second homework assignment that we are currently working on that is building our own portfolio. For this assignment we were to build all the code ourselves, none was provided for us. We were given a user story that explained what was being asked of us, a deployed portfolio of work samples set up in an easily accessible way for future employers to review but show the work we are capable of to decide if we are a good fit for the company. The acceptance criteria for this assignemnt was that there was a name, recent photo and links to the authors work as well as a contact form. Additionally we were to set up the page so that when you click on a link in the navigation the UI scrolls to the corresponding section, a section that shows the author's work, active links that take you to the deployed website for that body of work, images in the code and a responsive layout that adapts to a viewport.
